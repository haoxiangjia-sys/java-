// 1. 定义“类” (Class)
class GamePlayer {
    // --- 属性 (Attributes) ---
    String name;
    int health;

    // --- 构造函数 (Constructor) ---
    // 名字必须和类名完全一致，没有返回类型
    // 当你 "new" 的时候，它会被自动调用
    public GamePlayer(String inputName, int inputHealth) {
        System.out.println("正在根据图纸生产玩家对象...");
        this.name = inputName;    // 初始化属性
        this.health = inputHealth;
    }

    // --- 方法 (Methods) ---
    void takeDamage(int amount) {
        this.health -= amount;
        System.out.println(name + " 失去了 " + amount + " 点血量。");
    }
}

public class Main {
    public static void main(String[] args) {
        // 2. 实例化 (Instantiation)
        // 使用 new 关键字，触发构造函数
        GamePlayer player1 = new GamePlayer("阿尔法狗", 100); 
        GamePlayer player2 = new GamePlayer("柯洁", 90);

        // 3. 操作对象
        player1.takeDamage(20); // 调用方法
        System.out.println(player2.name + " 的当前血量: " + player2.health); // 访问属性
    }
}

5. minimax() → Core AI Algorithm
private double minimax(GameState state, int depth, double alpha, double beta, int mrXLocation)

Key terms you must know:
Minimax: a decision algorithm for two-player games
MAX player: MrX → wants to maximise the score
MIN player: Detectives → want to minimise the score
Depth: how many steps left to search
Alpha: best score for MAX (MrX)
Beta: best score for MIN (Detectives)

There are 3 stopping conditions:
Game over → return very high / very low score
Depth = 0 → use heuristic score (scoreBoard)
Out of time → return heuristic score

6. Alpha-Beta Pruning
if (beta <= alpha) break;

Pruning: means stop searching unnecessary branches
If alpha >= beta, this path cannot be better than previous options
Makes AI much faster

7. scoreBoard() → Heuristic Evaluation Function
private double scoreBoard(Board board, int mrXLocation)

This is the heuristic function (evaluation function).
It calculates how good the position is for MrX.Higher score = better for MrX.
It uses 3 factors:
Distance to the nearest detective (most important)
Sum of distances to all detectives
Freedom of movement (node connectivity)
This tells the AI if a position is safe or dangerous.
