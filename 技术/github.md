把这个 GitHub 仓库的代码同步到 VS Code
例如：git clone https://github.com/haoxiangjia-sys/java-.git


同步
初始化本地仓库：
git init
将所有文件添加到暂存区：
git add .
提交到本地仓库：
git commit -m "name"
关联远程 GitHub 仓库
git remote add origin https
推送
git branch -M main
git push -u origin main


cd .. 返回上一级
cd ~ 回到户主目录
cd / 回到根目录
pwd （print working direcrory）
