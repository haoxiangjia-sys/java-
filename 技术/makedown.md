# 📝 Markdown 语法实战笔记

这是一份专为 GitHub 学习设计的 Markdown 对照表。你可以通过此笔记快速掌握如何排版漂亮的技术文档。

---

## 1. 标题 (Headings)

**知识点说明**：
使用 `#` 加上一个**空格**来表示标题。`#` 越多，标题层级越低（字越小）。

**📌 代码示例：**
# 一级标题：用于文档大标题
## 二级标题：用于章节名称
### 三级标题：用于小节说明

---

## 2. 列表 (Lists)

**知识点说明**：
* **无序列表**：使用 `*`、`+` 或 `-` 加上空格。
* **有序列表**：直接使用数字加点 `1.`。

**📌 代码示例：**
* Git 基础操作
* Markdown 语法
* Java 环境搭建

1.  第一步：`git add .`
2.  第二步：`git commit -m "说明"`
3.  第三步：`git push`

---

## 3. 代码格式化 (Code)

**知识点说明**：
* **行内代码**：用两个反引号 `` ` `` 包裹，适合放在句子中间。
* **代码块**：用成对的三个反引号 ``` 包裹，可以在开头注明语言名称。

**📌 代码示例：**
在终端输入 `pwd` 可以查看当前所在的文件夹路径。

# 这是一个 Bash 代码块
git status
git add .
git commit -m "更新笔记"
