# 1. **证明** $n^2 + 10n + 8 \in O(\frac{1}{2}n^2)$
我们要找到 $c$ 和 $n_0$ 使得 $n^2 + 10n + 8 \le c \cdot (\frac{1}{2}n^2)$。

* **策略：** 当 $n \ge 1$ 时，我们可以通过放大各项来简化左式。
    * $n^2 = 1 \cdot n^2$
    * $10n \le 10n^2$ (当 $n \ge 1$)
    * $8 \le 8n^2$ (当 $n \ge 1$)
* **合并：** $n^2 + 10n + 8 \le 1n^2 + 10n^2 + 8n^2 = 19n^2$。
* **求解 $c$：** 我们需要 $19n^2 \le c \cdot \frac{1}{2}n^2$，即 $19 \le \frac{1}{2}c$，解得 $c = 38$。
* **结论：** 选择 $c = 38, n_0 = 1$。对于所有 $n \ge 1$，$n^2 + 10n + 8 \le 38(\frac{1}{2}n^2)$ 成立。

# 2. **证明** $n^3 + n^2 + n = O(n^3)$
我们要找到 $c$ 和 $n_0$ 使得 $n^3 + n^2 + n \le c \cdot n^3$。
* **策略：** 当 $n \ge 1$ 时：
    * $n^3 = 1 \cdot n^3$
    * $n^2 \le n^3$
    * $n \le n^3$
* **合并：** $n^3 + n^2 + n \le 1n^3 + 1n^3 + 1n^3 = 3n^3$。
* **结论：** 选择 $c = 3, n_0 = 1$。对于所有 $n \ge 1$，$n^3 + n^2 + n \le 3n^3$ 成立。

# 3. **证明** $10 \in O(1)$
我们要找到 $c$ 和 $n_0$ 使得 $10 \le c \cdot 1$。
* **策略：** 这非常直接，因为左边是一个常数。
* **结论：** 选择 $c = 10, n_0 = 1$。对于所有 $n \ge 1$，$10 \le 10$ 显然成立。

# 4. **证明** $\sum_{i=1}^{n} i \in O(4n^2)$
我们要找到 $c$ 和 $n_0$ 使得 $\sum_{i=1}^{n} i \le c \cdot 4n^2$。

* **第一步：使用等差数列求和公式**

$$\sum_{i=1}^{n} i = \frac{n(n+1)}{2} = \frac{1}{2}n^2 + \frac{1}{2}n$$

* **第二步：放大各项**
  当 $n \ge 1$ 时：
    * $\frac{1}{2}n^2 = \frac{1}{2}n^2$
    * $\frac{1}{2}n \le \frac{1}{2}n^2$
* **合并：** $\frac{1}{2}n^2 + \frac{1}{2}n \le 1n^2$。
* **求解 $c$：** 我们需要 $1n^2 \le c \cdot 4n^2$，即 $1 \le 4c$，解得 $c = 0.25$（或选更大的整数如 $c = 1$ 也可以）。
* **结论：** 选择 $c = 1, n_0 = 1$。对于所有 $n \ge 1$，$\sum_{i=1}^{n} i \le 1 \cdot 4n^2$ 成立。


## 2. Racetrack Principle
Use the racetrack principle to prove the following statement:
$$n \le e^n \text{ holds for every } n \ge 1 .$$

> **Solution.** First, we verify that $n \le e^n$ holds for $n = n_0 = 1$. This is true, since $1 \le e$ holds. 
> 
> Next, we verify that $(n)' \le (e^n)'$ holds for every $n \ge n_0$. We have $(n)' = 1$ and $(e^n)' = e^n$. We thus need to show that $1 \le e^n$ holds for every $n \ge 1$. 
> 
> Taking the natural logarithm on both sides, we obtain $0 \le n$, which is true for every $n \ge n_0 = 1$. Hence, $n \le e^n$ holds for every $n \ge 1$. $\checkmark$


## 3. O-notation: Part II
Give formal proofs of the following statements using the definition of Big-O from the lecture.

# 1. $f \in O(h_1), g \in O(h_2)$ then $f \cdot g \in O(h_1 \cdot h_2)$ .

**Solution.** Similar as in the previous exercise, we know that there are constants $c_1, c_2, n_1, n_2$ such that $f(n) \le c_1 \cdot h_1(n)$, for every $n \ge n_1$, and $g(n) \le c_2 \cdot h_2(n)$, for every $n \ge n_2$. Then:
$$f(n) \cdot g(n) \le c_1 \cdot h_1(n) \cdot c_2 \cdot h_2(n) = c_1 c_2 \cdot h_1(n)h_2(n)$$
for every $n \ge \max\{n_1, n_2\}$. We thus select $C = c_1 \cdot c_2$ and $N = \max\{n_1, n_2\}$ and obtain $f(n)g(n) \le C(h_1(n)h_2(n))$, for every $n \ge N$. $\checkmark$

# 2. $2^n \in O(n!)$ .

**Solution.** To prove this statement, we will show that $2^n \le C \cdot n!$ holds for $C = 2$ and every $n \ge 2$. To this end, observe that $2^n \le 2n!$ is equivalent to $2^{n-1} \le n!$. Observe that
$$2^{n-1} = \underbrace{2 \cdot 2 \cdot \dots \cdot 2}_{(n-1) \text{ times}},$$
and
$$n! = \underbrace{2 \cdot 3 \cdot \dots \cdot n}_{(n-1) \text{ factors, each larger equal to 2}}.$$
Trading off the factors of the two expressions, we see that $2^{n-1} \le n!$, which proves the result. $\checkmark$

---

# 3. $2^{\sqrt{\log n}} \in O(n)$ .

**Solution.** We need to show that there are constants $c, n_0$ such that $2^{\sqrt{\log n}} \le c \cdot n$ holds for every $n \ge n_0$. Observe that the previous inequality is equivalent to $2^{\sqrt{\log n}} \le 2^{\log(n) + \log(c)}$, which holds if $\sqrt{\log n} \le \log(n) + \log(c)$. Observe that $\sqrt{x} \le x$ for every $x \ge 1$. Hence, $\sqrt{\log n} \le \log(n)$ holds for every $\log(n) \ge 1$, or every $n \ge 2$. We can thus pick $n_0 = 2$ and $c = 1$ (observe that $\log(x) < 0$ for $x < 1$, we therefore couldn't choose $c < 1$). $\checkmark$
